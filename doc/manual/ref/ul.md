# Ul



Create an unordered list, just like in HTML



##  Child elements

[ForAll](../forall), [Li](../li)

##  Parent elements

[A](../a), [B](../b), [I](../i), [Li](../li), [Paragraph](../paragraph), [Span](../span), [Td](../td), [TextBlock](../textblock), [U](../u)


## Attributes



`class` (text, optional)
:   CSS class for this element.




`id` (text, optional)
:   CSS id for this element.




`style` (text, optional)
:   CSS style for this element.




## Example

```xml
<PlaceObject>
  <TextBlock>
    <Paragraph>
      <Ul>
        <Li><Value>First item</Value></Li>
        <Li><Value>Second item</Value></Li>
        <Li><Value>Third item</Value></Li>
      </Ul>
    </Paragraph>
  </TextBlock>
</PlaceObject>

```
