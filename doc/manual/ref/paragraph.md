# Paragraph



Insert a paragraph of text. The width of the paragraph is inherited from the surrounding element.



##  Child elements

[A](../a), [Action](../action), [B](../b), [Br](../br), [CopyOf](../copyof), [HTML](../html), [I](../i), [Ol](../ol), [Span](../span), [U](../u), [Ul](../ul), [Value](../value)

##  Parent elements

[Case](../case), [ForAll](../forall), [Loop](../loop), [Otherwise](../otherwise), [SetVariable](../setvariable), [Td](../td), [Textblock](../textblock), [Until](../until), [While](../while)


## Attributes



`class` (text, optional)
:   CSS class for this element.




`id` (text, optional)
:   CSS id for this element.




`style` (text, optional)
:   CSS style for this element.




## Example

```xml
<Textblock>
  <Paragraph>
    <Value>Hello World</Value>
  </Paragraph>
</Textblock>

```
