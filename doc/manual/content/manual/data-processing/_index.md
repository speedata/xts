# Data Processing

XTS is fundamentally a data-driven system: you feed it XML and it turns that data into PDF pages. This chapter covers how to structure your data, iterate over it, make decisions, use variables, and write custom functions.

- [Data Files](data-files) -- How to structure your XML data
- [Loops and Conditions](loops-and-conditions) -- ForAll, Loop, While, Until, Switch/Case
- [Variables and Functions](variables-and-functions) -- SetVariable, custom functions, data structures
- [XPath](xpath) -- XPath expressions and XTS layout functions
