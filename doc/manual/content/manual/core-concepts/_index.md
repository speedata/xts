# Core Concepts

Before you start building real layouts, it helps to understand the four pillars of XTS:

1. [**The layout-data pipeline**](how-it-works) -- how layout files and data files work together
2. [**The grid**](grid) -- how XTS positions objects on the page
3. [**Placing objects**](placing-objects) -- putting images, text, tables, and shapes on the page
4. [**Positioning areas**](positioning-areas) -- dividing pages into named regions
