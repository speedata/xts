# Tables

XTS provides a table model similar to HTML tables. If you've built a table in HTML, you'll feel right at home: rows with `<Tr>`, cells with `<Td>`, optional column definitions with `<Columns>`, and repeating headers with `<Tablehead>`.

Tables are one of the most powerful features in XTS -- they can span multiple pages with repeating headers, contain images and nested tables, and be styled with full CSS support.
